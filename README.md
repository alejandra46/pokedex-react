# Pokédex React

Aplicación desarrollada con React, TypeScript, Bootstrap y React Router que permite visualizar Pokémon, agregarlos a favoritos y consultar información detallada de cada uno.

## Funcionalidades

* Visualización de Pokémon.
* Agregar Pokémon a favoritos.
* Contador de favoritos usando Context API.
* Mostrar información del Pokémon favorito en el Navbar.
* Página de detalle para cada Pokémon.
* Navegación mediante React Router.

## Tecnologías utilizadas

* React
* TypeScript
* Vite
* Bootstrap
* React Router DOM
* Context API

## Instalación

```bash
npm install
npm run dev
```
